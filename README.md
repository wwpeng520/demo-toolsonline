##项目描述
该项目是个人项目，采用响应式布局方案，代码托管在coding.net上，旨在实现一个干净、完整的工具类站点。目前实现进制转换，分词和词频统计，常用加密解密实现，转义字符， HTML / CSS / JS格式化，天气查询，IP查询，用户客户端信息查询等功能。
